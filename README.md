# Open Plural (Backend)
> This is a Rust Rover project

## Requirements:
- `nightly` Rust toolchain
- MariaDB server

## Running the server:
1. Make sure you have a MariaDB server running and create a database for Open Plural.
2. Set your database credentials as environment variables: `DB_HOST`, `DB_PORT`, `DB_NAME`, `DB_USER`, and `DB_PASS`.
3. Run the server using `cargo run` in the project directory.
